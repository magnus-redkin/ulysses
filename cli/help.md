========================================================================================================
                                 ULYSSES VPN — СПРАВКА ОПЕРАТОРА CLI
========================================================================================================

КОМАНДА                     | ОПИСАНИЕ И КОНТЕКСТ ПРИМЕНЕНИЯ
========================================================================================================
uadmin system               | Мониторинг ресурсов сервера (Диск, RAM, Сервисы, ТОП процессов)
uadmin db reset             | Полный сброс базы данных с авто-наполнением тестовыми данными
uadmin db dump              | Создать дамп базы данных с ротацией и автоматической валидацией хвоста
uadmin db restore [файл]    | Безопасное восстановление контура из бэкапа без прав суперпользователя
uadmin db query "[sql]"     | Выполнить произвольный сырой SQL-запрос напрямую к таблицам биллинга
--------------------------------------------------------------------------------------------------------
uadmin user list            | Показать детальный список всех зарегистрированных клиентов
uadmin user create          | Создать пользователя вручную (флаги: --email, --tg-id, --username)
uadmin user delete          | Полное удаление аккаунта (БД + нода Hiddify) по ID/Email/TG-ID
uadmin user clean           | Массовое каскадное удаление всех тестовых аккаунтов (example / internal)
--------------------------------------------------------------------------------------------------------
uadmin sub list [query]     | Показать историю всех подписок (Умный фильтр по TG-ID/Email/UUID)
uadmin sub active           | Ручная выдача или кумулятивное продление подписки на --days
uadmin sub pending          | Имитация зависшей подписки (status=provisioning) для тестов биллинга
uadmin sub revoke           | Принудительный отзыв и блокировка всех активных подписок пользователя
--------------------------------------------------------------------------------------------------------
uadmin vpn link             | Сгенерировать персональную карточку и прямую ссылку доступа для Hiddify
uadmin vpn status           | Проверить конфигурацию и статус связи бэкенда с API Hiddify
--------------------------------------------------------------------------------------------------------
uadmin pay info [order_id]  | Сверить статус инвойса локально и запросить данные у шлюза эквайринга
uadmin stats                | Общая статистика. При аварии АВТОМАТИЧЕСКИ выводит список зависших подписок
uadmin fix pending          | Перезапустить обработку зависших подписок (протолкнуть в API ноды)
uadmin fix pending --force  | Сбросить счетчики ошибок (attempts=0) и принудительно активировать юзеров
uadmin fix sync             | Принудительная синхронизация состояний между локальной БД и Hiddify
uadmin fix invoices         | Очистка просроченных и неоплаченных счетов в биллинге
uadmin fix retry [sub_id]   | Точечная реанимация подписки (игнорирует active, если в Hiddify пусто)
========================================================================================================

Для вызова встроенной справки Click используйте: uadmin [модуль] --help
