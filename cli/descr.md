# Ulysses VPN — CLI-инструментарий администратора

## Назначение

`uadmin` — единая точка управления VPN-сервисом Ulysses Lab.
Объединяет команды для работы с базой данных, пользователями, подписками,
VPN-нодами, системным мониторингом и подсистемой защиты от блокировок (Мозг).

Вызов: `uadmin <модуль> <команда> [опции]`

## Архитектура

uadmin (cli/main.py)
├── system — мониторинг ресурсов сервера
├── db — управление базой данных
├── user — управление пользователями
├── sub — управление подписками
├── vpn — управление VPN-нодами (Hiddify)
├── stats — общая статистика
├── fix — восстановление и синхронизация
├── check — проверки и диагностика
├── notify — тестирование уведомлений
├── info — справка
└── brain — Мозг: мониторинг щитов и защита от блокировок (см. cli/brain/descr.md)

## Модули и команды

### system — мониторинг ресурсов сервера

Проверка диска, RAM, запущенных сервисов, TOP-процессов.
Используется для диагностики производительности сервера админки.

---

### db — управление базой данных

| Команда | Описание |
|---------|----------|
| `uadmin db reset` | Полный сброс базы данных с авто-наполнением тестовыми данными |

Вызывает `init_db.sh`, который пересоздаёт все таблицы (users, subscriptions,
payment_attempts, brain.*) и перезапускает systemd-сервис бэкенда.

---

### user — управление пользователями

| Команда | Описание |
|---------|----------|
| `uadmin user list` | Показать детальный список всех зарегистрированных клиентов |
| `uadmin user create` | Создать пользователя вручную (флаги: --email, --tg-id, --username) |
| `uadmin user delete` | Полное удаление аккаунта (БД + нода Hiddify) по ID/Email/TG-ID |

---

### sub — управление подписками

| Команда | Описание |
|---------|----------|
| `uadmin sub list` | Показать историю и текущий статус всех выписанных подписок |
| `uadmin sub active` | Ручная выдача или кумулятивное продление подписки на --days |
| `uadmin sub pending` | Имитация зависшей подписки (status=provisioning) для тестов биллинга |
| `uadmin sub revoke` | Принудительный отзыв и блокировка всех активных подписок пользователя |

---

### vpn — управление VPN-нодами

| Команда | Описание |
|---------|----------|
| `uadmin vpn link` | Сгенерировать персональную карточку и прямую ссылку доступа для Hiddify |
| `uadmin vpn status` | Проверить конфигурацию и статус связи бэкенда с API Hiddify |

Работает с Hiddify Manager через API. Использует `HIDDIFY_API_URL` и `HIDDIFY_API_KEY` из `.env`.

---

### stats — общая статистика

Статистика по пользователям, подпискам, платежам.
При аварии автоматически выводит список зависших подписок (status=provisioning).

---

### fix — восстановление и синхронизация

| Команда | Описание |
|---------|----------|
| `uadmin fix pending` | Перезапустить обработку зависших подписок (протолкнуть в API ноды) |
| `uadmin fix pending --force` | Сбросить счетчики ошибок (attempts=0) и принудительно активировать юзеров |
| `uadmin fix sync` | Принудительная синхронизация состояний между локальной БД и Hiddify |
| `uadmin fix invoices` | Очистка просроченных и неоплаченных счетов в биллинге |

---

### check — проверки и диагностика

Проверка целостности данных, связи с Hiddify API, статуса сервисов.

---

### notify — тестирование уведомлений

Отправка тестовых уведомлений (email, Telegram) для проверки каналов связи.

---

### info — справка

Выводит встроенную справку по всем модулям.

---

### brain — Мозг (защита от блокировок)

Полное описание: см. `cli/brain/descr.md`

Краткий список команд:

| Команда | Описание |
|---------|----------|
| `uadmin brain seed` | Заполнить brain-таблицы тестовыми данными |
| `uadmin brain telemetry` | Сгенерировать цикл телеметрии |
| `uadmin brain telemetry --anomaly` | Сымитировать блокировку |
| `uadmin brain detect` | Анализ телеметрии: поиск аномалий |
| `uadmin brain shield status` | Состояние всех щитов |
| `uadmin brain reset` | Очистить brain-таблицы |
| `uadmin brain dns show` | Показать DNS-записи (в разработке) |
| `uadmin brain dns sync` | Синхронизировать с Cloudns API (в разработке) |

---

## Техническая реализация

- **Язык:** Python 3.12
- **Фреймворк CLI:** Click
- **Вывод:** Rich (таблицы, цвета)
- **База данных:** PostgreSQL, схема `public` для ядра, схема `brain` для Мозга
- **Подключение к БД (brain):** синхронный `psycopg2`
- **Подключение к БД (backend):** асинхронный SQLAlchemy
- **Конфигурация:** `.env` в корне проекта
- **VPN-ядро:** Hiddify Manager (API)

## Переменные окружения (.env)

| Переменная | Назначение |
|------------|------------|
| `DB_HOST`, `DB_PORT`, `DB_NAME`, `DB_USER`, `DB_PASS` | Подключение к PostgreSQL |
| `HIDDIFY_API_URL`, `HIDDIFY_API_KEY` | API Hiddify Manager |
| `TELEGRAM_BOT_TOKEN`, `TELEGRAM_CHAT_ID` | Уведомления в Telegram |
| `SMTP_*` | Почтовые уведомления |
| `CLOUDNS_API_KEY`, `CLOUDNS_API_SECRET` | Управление DNS (в разработке) |
