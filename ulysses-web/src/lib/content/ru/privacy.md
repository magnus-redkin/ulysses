# Политика конфиденциальности

Политика конфиденциальности регулирует сбор, использование и защиту информации пользователей сервиса «Лаборатория Улисс» (далее — «Сервис»). Сервис собирает только те данные, которые необходимы для исполнения договора и предоставления цифровых услуг/товаров, включая идентификатор Telegram (ID), имя пользователя (username), адрес электронной почты и данные о платежных транзакциях, передаваемые через оператора фискальных данных. Данные используются для исполнения обязательств, обработки заказов и платежей, связи с Пользователем и предоставления поддержки. Передача информации третьим лицам осуществляется только в объёме и случаях, необходимых для исполнения договора (в том числе с платёжными/фискальными операторами) либо в предусмотренных законом случаях. Хранение данных осуществляется в течение срока, необходимого для целей обработки, а защита — в разумных пределах. Пользователь самостоятельно несёт ответственность за риски, связанные с передачей данных в рамках использования Сервиса. Администрация вправе вносить изменения в Политику без отдельного уведомления; согласие считается принятым при дальнейшем использовании Сервиса.

## Общие положения

1.1. Настоящая Политика конфиденциальности (далее — «Политика») регулирует порядок обработки и защиты информации, которую Пользователь предоставляет при использовании Сервиса.

1.2. Используя Сервис, Пользователь подтверждает согласие с условиями Политики. Если Пользователь не согласен с Политикой — он обязан прекратить использование Сервиса.

## Сбор информации

2.1. Сервис может собирать и обрабатывать следующие данные, необходимые для исполнения договора:

- идентификатор Telegram (ID);
- имя пользователя (username);
- адрес электронной почты;
- данные о платежных транзакциях (в объёме, необходимом для обработки платежей), передаваемые через оператора фискальных данных.

2.2. Сервис не требует от Пользователя предоставления паспортных данных, документов, фотографий или иной информации, не относящейся к целям исполнения договора.

## Использование информации

3.1. Сервис использует полученные данные исключительно для:

- исполнения обязательств по договору (предоставление цифровых услуг/товаров);
- обработки заказов и платежей;
- формирования и передачи платёжных/фискальных данных через соответствующего оператора;
- связи с Пользователем (в том числе для уведомлений и поддержки);
- урегулирования вопросов по заказам, доступу и оплате, включая рассмотрение заявок на возврат в соответствии с условиями Оферты.

## Передача информации третьим лицам

4.1. Администрация не передаёт данные третьим лицам, за исключением случаев, когда это необходимо:

- для исполнения обязательств по договору (в том числе платёжным и фискальным инфраструктурам/оператору фискальных данных);
- для защиты прав и законных интересов Администрации (при наличии законных оснований);
- в предусмотренных законом случаях.

## Хранение и защита данных

5.1. Данные хранятся в течение срока, необходимого для достижения целей обработки, исполнения обязательств по договору и соблюдения применимых требований законодательства.

5.2. Администрация принимает разумные меры для защиты данных от неправомерного доступа, изменения, раскрытия или уничтожения. При этом Администрация не гарантирует абсолютную безопасность передаваемой информации при использовании интернета.

## Отказ от ответственности

6.1. Пользователь понимает, что использование интернета и передача данных всегда сопряжены с рисками, включая риски, возникающие по причинам, не зависящим от Администрации.

6.2. Администрация не несёт ответственности за утрату, искажение или раскрытие данных, если это произошло по вине третьих лиц или вследствие действий/бездействия Пользователя.

## Изменения в Политике

7.1. Администрация вправе изменять условия Политики без предварительного уведомления.

7.2. Продолжение использования Сервиса после внесения изменений означает согласие Пользователя с новой редакцией Политики.
