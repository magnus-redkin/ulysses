# Privacy Policy

The data are used to fulfill the agreement, process orders and payments, communicate with the User and provide support. Disclosure to third parties is allowed only when required by law or to the extent necessary to perform obligations to the User (including through payment/fiscal infrastructure and the fiscal data operator). Data are stored for the period necessary to achieve the processing purposes, and protected to a reasonable extent. The User is solely responsible for risks associated with data transmission in connection with using the Service. The Provider may update this Privacy Policy without separate notice; continued use of the Service constitutes acceptance of the updated Policy.

1.1. This Privacy Policy (the “Policy”) regulates the processing and protection of information that the User provides when using the Service.

1.2. By using the Service, the User confirms their consent to the terms of the Policy. If the User does not agree with the Policy, the User must stop using the Service.

2.1. The Service may collect and process the following categories of data necessary to fulfill the agreement:

- email address;
- payment transaction data (within the scope required for payment processing), transmitted via the fiscal data operator.

2.2. The Service does not require the User to provide passport details, documents, photographs, or other personal information not related to the purposes of fulfilling the agreement.

## Use of information

3.1. The Service uses the obtained data exclusively for:

- fulfilling obligations under the agreement (provision of digital services/goods);

4.1. The Provider does not disclose the collected data to third parties except:
