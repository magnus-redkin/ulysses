КОМАНДА                         | ОПИСАНИЕ И КОНТЕКСТ ПРИМЕНЕНИЯ
--------------------------------------------------------------------------------------------------------
uadmin brain seed               | Заполнить brain-таблицы тестовыми данными (щиты + телеметрия)
uadmin brain seed --clear       | Очистить таблицы brain перед заполнением
uadmin brain telemetry          | Сгенерировать один цикл телеметрии для всех активных щитов
uadmin brain telemetry --anomaly| Сымитировать блокировку (0 клиентов, много ошибок)
uadmin brain telemetry --shield-id N | Сгенерировать телеметрию только для щита с ID=N
uadmin brain detect             | Анализ телеметрии: поиск аномалий (блокировок)
uadmin brain detect --shield-id N | Проверить только щит N
uadmin brain shield status      | Показать состояние всех щитов (IP, страна, DC, статус)
uadmin brain shield block N     | Вручную пометить щит N как заблокированный
uadmin brain shield unblock N   | Разблокировать щит N (перевод в резерв)
uadmin brain dns show           | Показать текущий DNS-кэш (brain.dns_state)
uadmin brain dns sync           | Синхронизировать DNS-кэш с Cloudns API (в разработке)
uadmin brain reset              | Полная очистка brain-таблиц (с авто-дампом перед сбросом)
--------------------------------------------------------------------------------------------------------
uadmin db dump                  | Создать дамп БД с проверкой целостности (ротация: latest + previous)
uadmin db restore <файл>        | Восстановить БД из дампа
--------------------------------------------------------------------------------------------------------

ТЕКУЩИЙ СТАТУС:
✓ seed, telemetry, detect, shield status, reset — работают
○ shield block/unblock — в разработке

ТИПОВЫЕ СЦЕНАРИИ ОТЛАДКИ:

1. Полный сброс и заполнение тестовыми данными:
   uadmin brain reset
   uadmin brain seed

2. Нормальный цикл + проверка:
   uadmin brain telemetry
   uadmin brain detect

3. Симуляция блокировки щита 1:
   uadmin brain telemetry --shield-id 1 --anomaly
   uadmin brain detect

4. Проверка одного щита:
   uadmin brain detect --shield-id 1
