## Development of Specialized Software

Design and deployment of proprietary software for automating IT monitoring, rapid smart-contract audits, and digital asset protection. Possible software solutions in the laboratory lineup:

- Ulysses Contract Scanner — modular software for automatic detection of vulnerabilities, hidden fees, and malicious code in smart contracts (Web3).
- Ulysses Node Monitor — server software for continuous monitoring of network infrastructure stability and recording of anomalous activity.
- Ulysses Analytical Parser — a data parser for collecting, aggregating, and structuring deep business analytics from foreign closed sources.
- Ulysses Access Bot Core — an off-the-shelf software solution for entrepreneurs that enables creating a subscription system and payment gateways inside Telegram.
